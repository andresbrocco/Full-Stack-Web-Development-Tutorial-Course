Specificity in CSS

When applying styles to elements, the browser calculates a "level of importance" to each style attribution, so in cases where there are more than one style attribution to the same element, it can decide which one should be applied.

There are four "levels" of specificity:

Importance
!important (Hard override)infinite
Style attribute (inside html)1000
ID100
Class, Pseudo-Class, Attribute10
Elements1

The total score is the sum of the importances given to that style. Examples:

  • h1 {style here} = 1 point
  • div h1 {style here} = 2 points
  • html h1 {style here} = 2 points
  • body h1 {style here} = 2 points
  • .aClassName h1 {style here} = 11 points
  • #anIDName {style here} = 100 points
  • [id="anIDName"] {style here} = 10 points
  • h1 {color: blue !important} = infinite points

Obs.:

  • [id="anIDName"] is a way to select attributes, so we are downgrading the importance of an ID to the importance of an attribute.
  • Hard overriding is highly not advisable.
  • If after the calculation there is a draw on the score, the last style found in the file is applied.
